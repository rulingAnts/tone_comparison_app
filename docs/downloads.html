Verifying Your Download

For security, you can verify your downloaded files using SHA256 checksums. Visit the v2.1.0 release page for checksums.

Windows

certutil -hashfile "Tone Matching Bundler-2.1.0-win-arm64.exe" SHA256

macOS/Linux

shasum -a 256 "Tone Matching Bundler-2.1.0-mac-universal.dmg"
