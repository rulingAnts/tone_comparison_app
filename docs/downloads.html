Verifying Your Download

For security, you can verify that your downloaded file matches the official release using the SHA256 checksum:

Windows

certutil -hashfile "Tone Matching Bundler Setup 1.0.0.exe" SHA256

macOS/Linux

shasum -a 256 "Tone Matching Bundler-1.0.0-universal.dmg"

Compare the output with the SHA256 values listed above. They should match exactly.
