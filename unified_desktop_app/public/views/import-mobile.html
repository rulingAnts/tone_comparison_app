Under Construction

This feature is currently under development and will be available in a future release.

Purpose: Sync changes made on mobile devices back to your desktop workflow.

Planned Workflow

Step 1: Export a bundle to mobile device using Export to Mobile tab

Step 2: User performs tone sorting/matching on mobile device (Android app)

Step 3: Import updated bundle from mobile device using this tab

Step 4: Compare changes and selectively merge updates into your working data

Planned Features

  • Import .tnset bundles from mobile devices
  • Visual comparison of changes (similar to comparison_app)
  • Selective merge of tone assignments and metadata
  • Conflict resolution for records modified in both places
  • Preserve tone group associations
  • Audit trail of changes made on mobile
  • Batch accept/reject changes by category

Technical Notes

This feature will incorporate functionality from the existing comparison_app, adapted for the unified workflow. The comparison engine code is available and can be integrated when development of this feature begins.
